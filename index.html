Spreadsheet

Spreadsheet Help

Features

  • 26 columns (A–Z) and 100 rows with sticky headers
  • Click or arrow keys to select cells; type to edit
  • Formula bar for values and formulas (starts with =)
  • Formulas: SUM, AVERAGE, MIN, MAX, and basic math
  • Drag the right edge of a column header to resize it
  • Format selected cells with the Size dropdown and Bold button
  • Select multiple cells by dragging or Shift+click / Shift+arrow keys
  • Copy, cut, and paste cell ranges with Ctrl+C, Ctrl+X, and Ctrl+V
  • Dependent cells recalculate automatically when inputs change
  • Save / Load to browser storage; Export / Import JSON files
  • Undo and redo for cell edits

Examples

Try these in your sheet:

Cell Enter this Result
A1 10 Plain number
A2 20 Plain number
A3 =SUM(A1:A2) 30
B1 =A1+A2 30
B2 =AVERAGE(A1:A2) 15
B3 =A1*2 20

Formulas must start with =. Use cell references like A1 and ranges like A1:A5.

Support

  • Runs entirely in your browser — no account or server required
  • Use Save to store your workbook in this browser; Load restores it
  • Use Export to download a JSON backup; Import to open one
  • Formula errors show in red: #REF! (bad reference), #ERROR! (invalid formula), #CYCLE! (circular reference)
  • Run locally with start.bat or npm run serve, then open http://localhost:3000
